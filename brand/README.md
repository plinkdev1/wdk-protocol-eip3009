# WDK · EIP-3009 — brand assets

This module is part of the **WDK** ecosystem, so its mark is the shared WDK
master badge (the **W** squircle, byte-identical to the wallet/checkout repos)
paired with an **"EIP-3009"** wordmark set in Zilla Slab Bold — a sibling of the
"WDK Wallet" / "WDK Pay" lockups.

| File | Use |
|---|---|
| `eip3009-lockup.svg` | Primary self-contained vector lockup (dark panel + badge + wordmark). |
| `eip3009-lockup-dark.png` / `-dark-1024.png` | Raster hero on a rounded dark panel; safe on any background. |
| `eip3009-lockup-ondark.png` / `-onlight.png` | Transparent variants (cream wordmark for dark / dark wordmark for light). |
| `eip3009-wordmark-{cream,dark}.svg` | Wordmark only. |
| `wdk-master-mark-{32…1024}.png` | The Bold **W** master badge. |
| `favicon.ico` | Multi-size favicon (the badge). |

Colours: WDK Orange `#FB451E`, Warm Dark `#15010C`, Cream `#F7EEE8`. The wordmark
ships outlined (no runtime font dependency); Zilla Slab is SIL OFL 1.1.
